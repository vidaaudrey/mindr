# Mindr

Tinder for Movies using React and Redux. 
Check more on [Wiki](docs/HOME.md)



## Installation

```bash
npm install
```

## Running Dev Server

```bash
npm run dev
```

## Building and Running Production Server

```bash
npm run build
npm run start
```
